## 🛠 The Tech Stack
We prioritized developer velocity and modern UI/UX standards using a React-based ecosystem.

*   **Frontend Framework:** [React](https://react.dev/) + [Vite](https://vitejs.dev/) (for lightning-fast HMR and build times).
*   **Styling:** [Tailwind CSS](https://tailwindcss.com/) for utility-first, responsive design.
*   **State Management:** React Context API (for global session management and real-time state updates).
*   **Routing:** React Router DOM (client-side SPA routing).
*   **Icons:** [Lucide React](https://lucide.dev/) (clean, consistent iconography).
*   **Data Handling:** Structured JSON mock-database with dynamic React `useState` hooks for real-time CRUD operations.

## 🤖 The AI Development Pipeline
This project was architected using a modular AI workflow to maximize productivity and code quality:

1.  **Full-Stack Scaffolding ([Bolt.new](https://bolt.new/)):** Used to bootstrap the application structure, establish routing, and configure the initial build environment in one prompt.
2.  **Logic Engineering ([Cursor](https://cursor.sh/)):** Utilized **Cursor Composer** for deep code refactoring, implementing complex array filtering logic, and managing state mutations.

## 🏗 Key Architecture & Features
*   **Two-Sided Marketplace:** Seamless role-switching between **Customer** (Booking/Discovery) and **Partner** (Dashboard/Menu Management).
*   **Dynamic State Engine:** Any new salon or service added via the "Partner Onboarding" flow instantly updates the global app state and renders in the search results without page refreshes.
*   **Real-time Filters:** JavaScript-powered filtering system that dynamically sorts salons by price, rating, and neighborhood.
*   **Responsive Design:** Mobile-first layout, ensuring a seamless experience across devices.

## 🚀 Deployment
Deployed as a high-performance Single Page Application (SPA).

*   **CI/CD:** Automatic deployments triggered via GitHub push.
